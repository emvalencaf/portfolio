-> texto sobre javascript
Uso o javascript para programar o front-end e o back-end (por meio do runtime do nodejs).

Meus conhecimentos sobre javascirpt consistem em manipulação da interface gráfica (DOM), iteirar arrays e objetos,classes, requisições assíncronas (AJAX), consumir API's, storage do navegador, manipulação de JSON e as features do ECMAScript2015.

Além da javascript, possuo conhecimento básico sobre typescript e sobre ferramentas como webpack e babeljs.



-> texto sobre o nodejs
Uso o nodeJS para executar códigos javascript fora do navegadores.

Meus conhecimentos sobre o nodeJS envolve o básico quanto a criação de servidores e rotas, por meio da ferramenta do ExpressJS, criação de API e criar conexões junto a banco de dados com o sistema do MongoDB.

-> texto sobre html5
Uso a linguagem HTML para criar a estrutura das webpages e dos webapp, usando sempre as marcações semânticas para facilitar a indexação do produto final pelos mecanismos de busca.

-> texto obre as css
Uso as CSS para aplicar estilos a estrutura das webpages e dos webapp. Usando essa tecnologia é possível exibir algumas animações, melhorar a acessibilidade do usuário e, é claro, tornar mais belo o layout do projeto.

-> git
O git é a ferramenta essencial para fazer backups dos projeto, controlar as suas versões. O que torna mais fácil o planejamento e introdução de novas features ao projeto.

-> photoshop
Uso o photoshop para manipular e tratar imagens, torna-las mais leves para ocupar menos espaço do servidor do projeto final.
